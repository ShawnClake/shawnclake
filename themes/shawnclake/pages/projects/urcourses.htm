Overview

UR Courses is a Course Management System used by the University of Regina.

UR Courses is integral to the University's online classes and provides a resource for in-person classes as well.

UR Courses is based off of the popular Moodle framework

While I was working with the Center for Continuing Education at the U of R, I was tasked with fixing bugs within the core Moodle framework and UR Courses, and actively developing and improving upon features such as: Course Mail, File Licensing, and Guides.

I was also directly involved with an incremental production upgrade process from Moodle 3.0 to 3.1

Feature Work
  • The File Licensing feature had to extend the existing licensing feature in Moodle. The system requires instructors to flag each uploaded file with a license. The licenses have to be customizable and not hard coded. As well, a 'Copyright Status Report' which details unlicensed files and licensed files must also be created.
